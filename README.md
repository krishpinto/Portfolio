# Krish Pinto's Portfolio

A minimal, pixel-perfect dev portfolio showcasing my work as a Software Engineer.

## Overview

### Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

### Featured

- Clean & modern design
- Light/Dark themes
- vCard integration
- SEO optimized

## Development

Install dependencies:
```bash
npm install
# or
pnpm install
```

Run the development server:
```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
